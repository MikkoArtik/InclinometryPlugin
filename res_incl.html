Результаты обработки инклинометрии

В результате обработки инклинометрии будут получены 7 физических файлов с расширениями *.cpg, *.dbf, *.prj, *.qpj, *.shp, *.shx и *.dat. Первые шесть — это векторный файл горизонтальной проекции ствола скважины, а последний — это файл с результатами расчетов, который выглядит так:

В шапке первая строка — это поправка за сближение меридианов, вторая — угол магнитного склонения (в градусах).
Далее основные данные в колонках:
MD — глубина по стволу;
Incl — зенитный угол (равен нулю при расчетах по способу DxDy);
Geo_Az — истинный азимут (равен нулю при расчетах по способу DxDy);
x_GKX — координата x в проекции Гаусса-Крюгера, где X — номер зоны;
y_GKX - координата y в проекции Гаусса-Крюгера, где X — номер зоны;
Vert_Depth — вертикальная глубина;
Altitude — абсолютная глубина.
Данный файл можно использовать для проверки корректности расчетов.
