  • Подготовка файла портов

    Для точек, чьи координаты и глубины необходимо получить, нужно подготовить входной файл следующего формата:
    - файл должен быть текстовый (формат *.dat);
    - в шапке две колонки (name — имя точки, MD — значение глубины по стволу, имена колонок могут быть другими);
    - разделитель между колонками — табуляция;
    - разделитель дробной части — точка.
