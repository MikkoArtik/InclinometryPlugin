Подготовка файла инклинометрии

Для начала нужно подготовить файл в текстовом формате с данными инклинометрии (рисунок 2). Необходимые колонки:
- в случае расчета по способу MD:
- Глубина по стволу (MD)
- зенитный угол (Incl)
- Азимут магнитный или истинный (Azim)

- в случае расчета по DxDy:
- смещение на Север/Юг (NS)
- смещение на Запад/Восток (WE)
- Вертикальная глубина (TVD)

Если предполагается расчет обоими способами, то в файле могут присутствовать все шесть колонок. В первой строке обязательно указываются названия колонок, а разделителем служит знак табуляции. Разделителем дробной части служит точка. При неверном формате файла обработка будет невозможной Файл должен быть сохранен в формате utf-8 с расширением *.dat. В файле не должно быть пустых строк.
