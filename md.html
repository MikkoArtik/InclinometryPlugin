Способ расчета по MD

Первым шагом расчета инклинометрии будет выбор типа расчета. В данном примере рассматриваем расчет по MD , поэтому его и выбираем в поле Processing type (#1). Помимо файла инклинометрии необходимы следующие входные данные:
- угол магнитного склонения (в случае, если в файле инклинометрии указан магнитный азимут. При истинном азимуте значение угла магнитного склонения равно 0);
- координаты устья;
- высота стола ротора;
- система координат, в которой даны координаты устья.

Далее нужно загрузить в плагин подготовленный файла инклинометрии (указать путь к файлу в поле Inclinometry file - #2).
Теперь указать колонки из файла, соответствующие смещениям глубине по стволу – MD, угол наклона – Inclination, азимут (магнитный или истинный) - Azimuth (#3). Задать необходимое значение магнитного азимута в поле Magnetic declination (#4). В поле Name (#5) вписываем имя скважины (эта надпись будет в названии файла, который мы получим по завершению).
В поле Coordinate system (#6) выбираем систему координат, в которой заданы координаты устья.
В полях x(WE), y(SN), Altitude (#7) указываются координаты устья и альтитуда стола ротора. В последнем поле Output folder (#8) выбирается папка для экспорта результатов. После заполнения всех нужных полей нажимаем кнопку «Apply».

При правильном вводе данных в папке из поля Output folder (#7) после расчета будут 7 файлов с результатами (6 – векторные файлы, 1 – текстовый). Полученный shp-файл автоматически подгружается в QGis.

Описание выходного формата см. в разделе Результат обработки (ссылка на раздел)
